Instrucciones de compilación en caso de ser necesarias, y la forma de compilación que usó (debe ser alguna de las indicadas en los 
tutoriales entregados en Aula USM).

En un computador con sistema operativo Linux, se instaló el compilador de C++ mediante el comando en la terminal "sudo apt-get gcc". 
Luego, se traspasó el código a un archivo llamado "tarea3-Rioseco-apellido2.cpp". Se accedió a la carpeta anterior, se ocupó el comando "g++ tarea3-Rioseco-apellido2.cpp -o tarea3-Rioseco-apellido2" para que se compilara, y luego el comando "./tarea3-Rioseco-apellido2". Por último, se le entregó una entrada al programa para que ejecutara.
